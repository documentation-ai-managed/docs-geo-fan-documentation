## Upgrading to the Latest Version

Follow these steps to upgrade GEO.fan and benefit from the newest features.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command:

    ```bash
    npx geo-fan --version
    ```
  </Step>
  <Step title="Update Package" icon="download">
    Use your preferred package manager:

    <Tabs>
      <Tab title="npm" icon="npm">
        ```bash
        npm update geo-fan@latest
        ```
      </Tab>
      <Tab title="yarn" icon="yarn">
        ```bash
        yarn upgrade geo-fan@latest
        ```
      </Tab>
      <Tab title="pnpm" icon="package">
        ```bash
        pnpm update geo-fan@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Rebuild Site" icon="refresh-cw">
    Rebuild your documentation:

    ```bash
    npx geo-fan build
    ```
  </Step>
  <Step title="Deploy Changes" icon="upload-cloud">
    Deploy to your hosting provider, such as Vercel or Netlify.
  </Step>
</Steps>

<Callout kind="tip">
  Backup your `docs/` folder before upgrading to avoid data loss.
</Callout>
